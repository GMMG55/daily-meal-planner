# Daily Meal Planner - 每日智能菜谱推荐 v1.0.1

智能推荐菜谱，提供热量、营养、食材和详细做法。支持用户画像记忆和偏好学习。

---

## ✨ 核心功能

### 🍽️ 智能三菜推荐

每餐推荐3套完整套餐（主菜+配菜+汤/主食）：

1. **📌 综合智能推荐** — 结合季节、周几、天气、用户画像加权评分
2. **🌿 时令之选** — 纯按当季食材筛选
3. **🎲 随机惊喜** — 换换口味也不错

### 👤 用户画像系统

自动收集并保存到本地（`user_profile.json`）：

| 维度 | 示例 | 作用 |
|---|---|---|
| 地点 | 北京、上海、成都 | 自动查天气、优先推荐本地风味 |
| 心情 | 开心/忙碌/疲惫/放松/庆祝/慵懒 | 心情→菜品匹配 |
| 想吃类别 | 主食/肉/素/汤/辣/清淡/甜/小食 | 类别→标签匹配 |
| 偏好菜系 | 川菜/粤菜/鲁菜/京味/江浙/东北 | 家乡味道加权 |
| 喜欢的菜 | 西红柿牛腩、清炒时蔬 | 永远优先推荐 |
| 不喜欢的菜 | 苦瓜、香菜 | 永远不会推荐 |
| 饮食目标 | 减肥/增肌/保持/养生 | 低卡/高蛋白/滋补模式 |
| 过敏食材 | 花生、海鲜 | 安全过滤 |

## 🚀 使用方法

### 基础命令

```bash
# 今日推荐（自动判断餐次）
python scripts/meal_recommend.py daily

# 指定餐次
python scripts/meal_recommend.py daily --meal 午餐
python scripts/meal_recommend.py daily -m 晚餐
python scripts/meal_recommend.py daily -m 早餐

# 支持英文餐次
python scripts/meal_recommend.py daily --meal breakfast
python scripts/meal_recommend.py daily --meal lunch
python scripts/meal_recommend.py daily --meal dinner

# 指定天气
python scripts/meal_recommend.py daily --weather rainy
python scripts/meal_recommend.py daily -w hot

# 一周菜单
python scripts/meal_recommend.py weekly
```

### 进阶用法

```bash
# 搜索菜品
python scripts/meal_recommend.py search 西红柿
python scripts/meal_recommend.py search 清淡

# 查看详细做法
python scripts/meal_recommend.py detail 西红柿牛腩
python scripts/meal_recommend.py detail 清炒时蔬

# 推荐更多
python scripts/meal_recommend.py daily -n 5
```

## 📊 输出格式

每道推荐包含：

```
  ────────────────────────────────
  ☀️ 推荐1  西红柿牛腩
  📝 酸甜浓郁肉烂汤鲜
  💡 应季食材，周五庆祝，晴天清爽
  🔥 650kcal  ⏱ 90min  难度: 中等
  📊 蛋白质:28g | 铁:4mg
  🏠 南方,江浙,粤菜
  🥗 食材: 牛腩400g,番茄3个,土豆1个,洋葱半个
  🥬 +配: 清炒时蔬（120kcal）
  🍲 +汤/饭: 白灼虾（150kcal）
  ────────────────────────────────
```

**字段说明**：
- 🍽️ 菜名 + 简介
- 💡 推荐理由（季节、周几、天气、心情、地域等）
- 🔥 总热量（主菜+配菜+汤）
- ⏱ 烹饪时间
- 📊 营养亮点（蛋白质、维C、铁、钙等）
- 🏠 地域标签
- 🥗 完整食材清单
- 🥬 配菜推荐
- 🍲 汤/主食搭配

## 🔧 技术细节

### 评分算法

```python
score = 季节匹配(+3)
      + 周几权重(工作日/周末)
      + 天气权重(清热/暖身/祛湿)
      + 心情权重(+1.5 per tag)
      + 类别权重(+1 per tag)
      + 喜欢的菜(+2)
      + 地域匹配(+2)
      + 随机微调(0~1.5)
```

### 模板位置

- 脚本：`scripts/meal_recommend.py`
- 数据：`scripts/meals_db.json`
- 画像：`scripts/user_profile.json`（自动生成）
- 搜索：`scripts/meal_search.py`（在线搜索下厨房/美食杰）
