## 季节饮食建议

- **春季**：养肝为主，多食绿色蔬菜、芽菜，少酸多甘
- **夏季**：清热解暑，多食瓜果、凉菜、绿豆汤，少油腻
- **秋季**：润燥养肺，多食银耳、梨、百合、蜂蜜
- **冬季**：温补御寒，多食牛羊肉、根茎类、炖汤

## 常见饮食搭配禁忌

- 菠菜 + 豆腐 → 影响钙吸收（焯水可去除大部分草酸）
- 螃蟹 + 柿子 → 可能引起不适
- 牛奶 + 巧克力 → 影响钙吸收
- 白酒 + 柿子 → 可能引起胃肠不适
